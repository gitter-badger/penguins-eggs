# V. 0.4.x (development)

Change paramenter

usage: egg <commands>

# options
```
egg                 help
egg distro  --create                  create egg [name] and netboot
            --distro [name]
            --user [user]
            --password [password]
egg distro   --rebuild [name] destroy and re-create egg and netbook

egg netboot --install   install netboot services
egg netboot --purge     remove and purge netboot services
egg netboot --start     start netboot services
egg netboot --stop      stop netbook services
egg netboot --restart   restart netboot services
egg --help          this help
```
